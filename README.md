# Компьютерная графика: Лабораторная работа №1

Данный проект представляет собой реализацию алгоритмов двумерных и трехмерных аффинных преобразований с использованием математического аппарата однородных координат.

**Вариант:** 16  
**Стек технологий:** Java 21, Maven, Java Swing, AWT (Graphics2D).

## Функционал

### 1. 2D Преобразования на плоскости
Смоделирован сложный вогнутый многоугольник (8 вершин). Реализована композиция следующих преобразований:
1. **Отражение** относительно оси OX.
2. **Перенос** на вектор `(4, -3)`.
3. **Поворот** на `225°`.
4. **Масштабирование** с коэффициентами `(0.9, 1.1)`.

**Особенности 2D-модуля:**
- Отображение статического "шлейфа" (промежуточных состояний) фигуры.
- Плавная кусочно-линейная интерполяция (анимация) процесса трансформации.
- Интерактивное управление камерой (панорамирование мышью, зум колесиком).

### 2. 3D Преобразования в пространстве
Смоделирована каркасная 3D-сцена, состоящая из параллелепипеда и пирамиды. Реализована композиция:
1. **Перенос** на вектор `(-2, 1, 3)`.
2. **Вращение Z** на `270°`.
3. **Масштаб** с коэффициентами `(1.3, 0.7, 1)`.
4. **Вращение Y** на `90°`.

**Особенности 3D-модуля:**
- Использование матриц `4x4` и однородной координаты `W`.
- Ортографическая проекция на плоскость экрана (отсечение координаты `Z`).
- Интерактивная 3D-камера для вращения сцены мышью.
- Вычисление детерминантов матриц для анализа изменения объема объектов.

## Архитектура проекта

Проект построен без использования сторонних математических библиотек (таких как JOML или Apache Commons Math). Все вычисления линейной алгебры реализованы с нуля:
- `Vector3` и `Vector4` — представления точек в однородных координатах.
- `Matrix3` и `Matrix4` — математические ядра с функциями генерации матриц трансформации, умножения (композиции) и вычисления детерминантов.

## Запуск проекта

Убедитесь, что у вас установлена **Java 21** или выше.

1. Клонируйте репозиторий:
   ```bash
   git clone [https://github.com/ВАШ_ЛОГИН/Graphics-Lab-1.git](https://github.com/ВАШ_ЛОГИН/Graphics-Lab-1.git)
   ```
2. Откройте проект в любой современной IDE (IntelliJ IDEA, Eclipse) как Maven-проект.

3. Запустите класс Main.java (находится в пакете ru.university.graphics).

4. В консоли выберите режим работы (1 для 2D, 2 для 3D). Вывод матриц и детерминантов также осуществляется в консоль среды разработки.
